Today, we released Quarkus 2.2.3.Final, a maintenance release for our 2.2 release train containing a few bugfixes and platform version bumps.

It is a safe upgrade for anyone already using 2.2.

If you are not using 2.2 already, please refer to the https://github.com/quarkusio/quarkus/wiki/Migration-Guide-2.2[2.2 migration guide].

== Full changelog

You can get https://github.com/quarkusio/quarkus/releases/tag/2.2.3.Final[the full changelog of 2.2.3.Final on GitHub].
