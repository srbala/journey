Today, we released 2.7.5.Final which, as usual, only contains bugfixes and documentation improvements.

It is a safe upgrade for anyone already using 2.7.

If you are not using 2.7 already, please refer to the https://github.com/quarkusio/quarkus/wiki/Migration-Guide-2.7[2.7 migration guide].

== Full changelog

You can get https://github.com/quarkusio/quarkus/releases/tag/2.7.5.Final[the full changelog of 2.7.5.Final on GitHub].
